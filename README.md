25 ÅR FEIRES MED 25 TIMERS MORO
===============================

21. - 22. juni utvider DNT ung døgnet til 25 timer for å feire sin egen 25-årsdag. Hver eneste av disse 25 timene vil en ny DNT ung-aktivitet starte rundt omkring i landet. 

## Funksjonalitet
* Interaktivt kart
